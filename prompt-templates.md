# Claude Design 指令範本

## 基本指令格式

```
使用我的 Design System，製作一份簡報。

簡報主題：[主題]
頁數：[頁數]

頁面清單：
Slide 1：封面頁，主標題「[標題]」，副標題「[副標題]」
Slide 2：[頁面類型]，[內容說明]
Slide 3：[頁面類型]，[內容說明]
```

---

## 範例指令 1：概念提案

```
使用我的 Design System，製作一份概念提案簡報。

簡報主題：2025 品牌重塑提案
頁數：8 頁

Slide 1：封面頁，版型A，主標題「Brand Renewal 2025」
Slide 2：章節分隔頁，「01 — 現況分析」
Slide 3：內容頁，版型A，標題「問題所在」，說明文字 40 字以內
Slide 4：數據頁，三個數字：-23% 品牌辨識度 / 41% 競爭者成長 / 2.1x 預算差距
Slide 5：章節分隔頁，「02 — 解決方向」
Slide 6：內容頁，版型C，關鍵字散落：Minimal / Bold / Consistent / Modern
Slide 7：內容頁，版型B，四張圖片佔位格，標題「視覺參考方向」
Slide 8：結尾頁，版型A，「Thank You」
```

---

## 範例指令 2：成效報告

```
使用我的 Design System，製作一份行銷成效報告。

Slide 1：封面頁，主標題「Q2 Campaign Report」
Slide 2：數據頁，單一大數字，「$3.2M」，標籤「總營收」
Slide 3：數據頁，三個數字：+18% 成長率 / 94% 客戶滿意度 / 2.4M 觸及人數
Slide 4：時間軸頁，標題「Campaign 時間軸」，節點：4/1 啟動 / 4/15 第一波 / 4/29 高峰 / 5/10 結束
Slide 5：結尾頁，版型A
```

---

## 注意事項

- 每頁只說一件事
- 說明文字控制在 40 字以內
- 數字頁不需要額外說明，數字本身就是訊息
