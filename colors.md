# Color Tokens

## 主色板

| Token | Hex | 用途 |
|-------|-----|------|
| `bg-primary` | `#0A0A0A` | 所有頁面背景 |
| `bg-surface` | `#2A2A2A` | 卡片、圖片佔位區塊 |
| `bg-subtle` | `#1A1A1A` | 次要區塊背景 |
| `text-primary` | `#FFFFFF` | 主標題、大字 |
| `text-secondary` | `#A0A0A0` | 內文、說明文字 |
| `text-muted` | `#555555` | caption、頁碼 |
| `accent` | `#A6A5C4` | 強調色、線條、重點字 |

## 使用規則

- 背景永遠是 `#0A0A0A`，不用純黑 `#000000`
- accent `#A6A5C4` 只用在關鍵數字、重點字、裝飾線條
- 不超過 3 個顏色同時出現在一頁
- 圖片區塊用 `#2A2A2A` 作佔位底色

## 禁止

- 不用白色背景
- 不用彩色背景
- 不用漸層
