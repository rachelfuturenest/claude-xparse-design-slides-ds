# 視覺元素規範

## 圖片格

- 佔位底色：`#2A2A2A`
- 圖片格不加邊框、不加圓角
- 圖片與圖片之間間距：0.3cm
- 圖片格比例：16:9 或 4:3，保持一致

## 線條

- 裝飾線：1px，顏色 `#A6A5C4`
- 分隔線：1px，顏色 `#2A2A2A`
- 時間軸線：1px，顏色 `#A6A5C4`

## 幾何形狀（謹慎使用）

- 只用矩形，不用圓形、三角形等其他形狀
- 顏色限定 `#2A2A2A` 或 `#A6A5C4`
- 不加陰影、不加漸層

## 禁止

- 不用 icon 圖示
- 不用插圖或 illustration
- 不加任何裝飾性元素
- 不用邊框框住文字
